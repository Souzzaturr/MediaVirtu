### O que é o MediaVirtu?

MediaVirtu é uma rede social de postagens de humor recheados de comédia. Nosso objetivo principal é tornar possível que nossos usuários possam trazer grandes sorrisos para todos que verem seus posts. Navegue por esse belíssimo site, descubra os mais incríveis e divertidos posts que você pode encontrar, divirta-se!!!


### A origem

Esse site iniciou como um projeto acadêmico criado pelos alunos [Artur da Silva Souza](https://ifpb.github.io/projects/people/202514320010/), [Josué Benedito dos Santos Neto](https://ifpb.github.io/projects/people/202514320003/) e [Matheus Roberto Lira Dias](https://ifpb.github.io/projects/people/202514320001/) para a disciplina de Programação para Web II, do curso de Engenharia de Software do [Instituto Federal da Paraíba](https://www.ifpb.edu.br/). Nesse site, foi implementado na prática o desenvolvimento de aplicações frontend, utilizando principalmente as linguagens de programação JavaScript e Typescript, o framework Next.js e a biblioteca React.
